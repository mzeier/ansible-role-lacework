Lacework Ansible Role
=========

Ansible Role to deploy the Lacework agent.


Platforms
---------

* Amazon Linux
* CentOS
* RedHat
* Ubuntu

Role Variables
--------------
The following variables are available for override.
```
lacework_accessToken: :         # Required. Your API Key
```


Dependencies
------------

None
